Includ

  

O projeto INCLUD teve como objetivo realizar a inclusão digital através do software livre utilizando como ferramentas o LTSP (GNU/Linux Terminal Server Project) executando sobre o sistema operacional GNU/Linux. O LTSP visa o reaproveitamento de computadores ultrapassados e terminais de baixo custo, provendo assim um ambiente de trabalho baseado em rede. O INCLUD desenvolveu uma ferramenta que permitiu gerenciar os processos gerados por cada terminal, gerar relatórios, parar terminais clientes, monitorar usuários e recursos no sistema, entre outros.

Tecnologias usadas: LTSP, Shell Script, PHP, Apache
